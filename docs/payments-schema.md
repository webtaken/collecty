## Detailed Feature Breakdown

### Core Features (All Plans)

#### Project Management
- Create and manage email collection projects
- Project settings (name, domain, description)
- Activate/deactivate projects instantly

#### Widget Types
- **Popup Widget**: Modal-style email capture
- **Inline Widget**: Embedded form directly on page
- Both widget types customizable for all plans

#### Smart Triggers
- **Delay**: Show widget after X seconds
- **Scroll**: Show when user scrolls to X%
- **Exit Intent**: Detect when user is about to leave
- **Click**: Trigger via manual button click
- All triggers available on all plans

#### Widget Customization
- Custom colors (primary, background, text)
- Button text and styling
- Success messages
- Position options (center, corners)
- Placeholder text
- Border radius
- Layout options (horizontal/vertical for inline)

#### Lead Magnets
- Rich text descriptions (Tiptap editor)
- Preview text for widgets
- Attach lead magnets to any widget
- Download delivery after subscription

### Free Trial Features

#### Unlimited Everything
- No restrictions on projects, widgets, or subscribers
- Full API access with rate limits (Coming Soon)
- All integrations available (Coming Soon)
- Webhook support (Coming Soon)
- Advanced analytics (90-day retention) (Coming Soon)
- Priority support (Coming Soon)

#### Trial Benefits
- Full feature access for 15 days
- No credit card required
- Easy upgrade after trial
- Graceful downgrade to Hobby plan if not upgraded

### Hobby Plan Features

## Upgrade/Downgrade Paths

### Free Trial → Hobby
- Automatic after 15 days if no payment method added
- Keep up to 5 projects and 1000 subscribers
- Excess projects/subscribers archived
- API access disabled (Coming Soon)
- Webhooks removed (Coming Soon)

### Free Trial → Serious
- Upgrade anytime during trial
- All projects, widgets, subscribers retained
- Full feature access continues
- No interruption in service

### Hobby → Serious
- Instant upgrade
- All projects and widgets retained
- All subscribers retained
- Features unlock immediately
- Prorated billing

### Serious → Hobby
- Grace period: 7 days before data deletion
- Excess projects archived (keep first 5)
- Excess subscribers archived (keep first 1000)
- API keys deleted (Coming Soon)
- Webhooks removed (Coming Soon)
- Team members removed (Coming Soon)
